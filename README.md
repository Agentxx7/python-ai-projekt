# Python AI-projekt

Detta projekt är en inlämningsuppgift i kursen Pythonprogrammering för AI-utveckling.

## Syfte
Syftet med projektet är att skapa en enkel maskininlärningsmodell som kan avgöra om en kandidat matchar ett jobb eller inte.

## Filer
- `create_dataset.py` – skapar datasetet
- `analysis.py` – analyserar data, tränar modellen och utvärderar resultatet
- `job_match_dataset.csv` – datasetet
- `Namn Jonny Frez.pdf` – rapporten

## Metod
Projektet använder Python och Scikit-learn för att:
- läsa in data
- förbereda features
- dela upp data i tränings- och testdata
- träna en klassificeringsmodell
- utvärdera resultatet

## Körning
Kör projektet med:

```bash
python analysis.py
